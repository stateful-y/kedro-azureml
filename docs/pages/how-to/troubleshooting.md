# Troubleshooting

Solutions to common problems when using Kedro AzureML Pipeline.

## Installation Issues

**Problem: Package not found**
: Verify the package name and that it is published: `pip install kedro_azureml_pipeline`

**Problem: Import error after installation**
: Make sure you installed in the active environment: `python -c "import kedro_azureml_pipeline"`

## Runtime Issues

**Problem: Unexpected behavior**
: Check your configuration and consult the [Configuration guide](configure.md).
